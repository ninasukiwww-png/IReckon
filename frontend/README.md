# IReckon Frontend 🌸

Vue3 现代前端界面 (๑•́ ∀ •̀๑)

---

## 📦 安装依赖 (先把东西装好！)

```bash
cd frontend
npm install
```

---

## 🛠️ 开发模式 (开始愉快的开发吧！)

```bash
npm run dev
```

然后访问 http://localhost:3000 就可以看到美美的界面啦 ✿

---

## 🏗️ 构建 (打包上线！)

```bash
npm run build
```

---

## ✨ 功能 (超多功能等你来玩！)

- 💬 **聊天界面** — 多层消息、实时 WebSocket，聊天超流畅！
- 📋 **任务管理** — 创建、取消、恢复任务，掌控全局！
- 📊 **仪表盘** — 系统状态、统计数据，一目了然！
- 🤖 **AI 实例管理** — 添加、编辑、删除、测试 AI 端点，超方便！
- ⚙️ **设置** — 系统配置查看，你想知道的都在这里！
- 🌙 **深色/浅色主题切换** — 白天夜晚随心换，超自由的！
